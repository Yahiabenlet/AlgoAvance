# Triangulation Minimale d’un Polygone

## Description
Ce projet permet de générer un polygone convexe irrégulier et de calculer une triangulation minimale en utilisant plusieurs algorithmes (backtracking, programmation dynamique, glouton).

## Installation

1. Assurez-vous d'avoir Python 3.8 ou supérieur installé.
2. Cloner ce dépôt ou télécharger les fichiers.
3. Installer les dépendances (ici uniquement des modules standard Python : `math` et `random`).

## Utilisation

Exécuter le programme principal directement depuis le terminal :

```bash
python3 ProjetAlgo.py
```
